# Coffee-Shop-Sales-Dashboard
Interactive Excel dashboard for retail sales analysis, featuring data cleaning, transformation, pivot tables, charts, slicers, and key metrics. Provides actionable insights on revenue, footfall, and product performance, with a well-structured dashboard for easy updates and reproducibility.

## Overview
This project is an interactive Excel dashboard designed to analyze retail sales data and provide meaningful business insights. It showcases data cleaning, transformation, and visualization to track sales performance effectively.

## Objective
To analyze retail sales data and generate actionable insights to improve overall business performance.

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Formulas
- Power Query
- Charts & Slicers

## Features
- Data cleaning and transformation for accurate analysis  
- Interactive dashboard with charts and slicers  
- Visualization of key metrics like revenue, footfall, and product performance  
- Analysis of sales trends and patterns
