# Coffee-Shop-Sales-Dashboard
Interactive Excel dashboard for retail sales analysis, featuring data cleaning, transformation, pivot tables, charts, slicers, and key metrics. Provides actionable insights on revenue, footfall, and product performance, with a well-structured dashboard for easy updates and reproducibility.


# Coffee Shop Sales Dashboard

## Objective
Analyze retail sales data to gain actionable insights and improve Coffee Shop performance.

## Tools & Technologies
- Excel (Pivot Tables, Formulas, Power Query, Charts, Slicers)

## Dataset
- 149K+ rows of retail sales data
- Includes transaction details, product information, sales amount, date, and store locations

## Key Features / Contributions
- **Data Cleaning & Transformation:** Handled large datasets, removed inconsistencies, and prepared data for analysis.
- **Dashboard Development:** Built interactive dashboards with charts, slicers, and calculated measures to visualize revenue, footfall, and product trends.
- **Sales Analysis & Insights:** Analyzed sales patterns and KPIs to identify trends and evaluate store and product performance.
- **Documentation:** Documented data sources, transformation logic, and dashboard structure for reproducibility and easy updates.
